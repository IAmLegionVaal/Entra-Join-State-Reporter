# Entra Join State Reporter

A read-only PowerShell toolkit for Microsoft Entra device join-state reporting.

## Features

- Azure AD, Entra hybrid, and workplace join context
- Device registration command output
- Domain and workgroup information
- TXT, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Entra_Join_State_Reporter.ps1
```

## Safety

Read-only reporting only. No device registration settings are changed.
